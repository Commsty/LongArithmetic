## Описание проекта

Простой учебный проект по реализации длинной арифметики на C++. Библиотека позволяет работать с очень большими числами, которые не помещаются в стандартные типы данных. Поддерживает как целые, так и дробные числа с настраиваемой точностью.

Проект писался в рамках изучения алгоритмов и структур данных — чтобы понять, как работают большие числа "под капотом".

## Что умеет библиотека

- **Работа с большими числами** — складывать, вычитать, умножать и делить числа, которые не помещаются в `long double`
- **Дробные числа** — поддержка десятичных дробей с заданной точностью
- **Современный C++** — использовал C++17, RAII, move-семантику
- **Тесты** — написал юнит-тесты для проверки всех операций
- **Пример** — есть программа для вычисления числа π

## Как это работает

Числа хранятся в `std::deque<unsigned long long>` — каждый элемент это 64-битный блок. Точность можно настроить отдельно для целой и дробной частей. Реализовал все основные арифметические операции "вручную" — столбиком, как в школе, только с большими числами.

## Как запустить

```bash
# Собрать библиотеку
make

# Запустить тесты
make test

# Посчитать число π (пример работы)
make pi
```

### Пример использования

```cpp
#include "Longnum.hpp"

int main() {
    // Создаем числа с точностью 100 знаков до и после запятой
    LongNumber a("123.456789", 100, 100);
    LongNumber b("987.654321", 100, 100);

    // Делаем операции
    LongNumber sum = a + b;
    LongNumber product = a * b;

    // Выводим результат
    std::cout << "Сумма: " << sum.GetDecimal() << std::endl;
    std::cout << "Произведение: " << product.GetDecimal() << std::endl;

    return 0;
}
```

## Что интересного в реализации

- **Хранение чисел** — использую `std::deque<unsigned long long>`, каждый элемент это 64-битный блок
- **Настраиваемая точность** — можно отдельно задать точность для целой и дробной частей
- **Move-семантика** — добавил перемещение для оптимизации производительности
- **Нормализация** — числа автоматически оптимизируются для экономии памяти

2. Реализовать в `Longnum.cpp`
3. Написать тест в `test.cpp`
